NeverDry

Irrigação inteligente para Home Assistant. Um modelo científico de balanço hídrico que sabe quando e por quanto tempo regar o seu jardim.

Repositório GitHub Documentação

Funcionalidades

🌡️

Modelo ET Científico

Balanço hídrico FAO-56 simplificado. Estima a evapotranspiração a partir da temperatura com dois parâmetros calibráveis.

🌱

10 Famílias de Plantas

Coeficiente de cultura (Kc) por zona com variação sazonal. Relva, hortaliças, suculentas, árvores de fruto e mais. Adaptação automática ao hemisfério.

🚿

Controlo Direto de Válvulas

Abre e fecha válvulas automaticamente. Irrigação multi-zona sequencial com paragem de emergência.

📊

Défice por Zona

Cada zona rastreia o seu próprio défice hídrico do solo. Plantas diferentes secam a ritmos diferentes — como deve ser.

🌧️

Sensível à Chuva

Salta automaticamente a rega em dias chuvosos. O défice diminui a cada evento de chuva.

⚙️

Configuração por Interface

Configuração completa pela interface do Home Assistant. Sem necessidade de editar YAML.

🔄

Atualizações Transparentes

Releases automatizadas via GitHub Actions. A migração de configuração preserva as definições entre versões. Atualização com um clique no HACS.

🖐️

Válvula manual reconhecida

Abra a válvula pelo botão físico, app Zigbee ou HA — o NeverDry segue a sessão, atualiza o défice e fecha automaticamente no mínimo entre água necessária e timeout de segurança.

💧

Todos os gatilhos cobertos

Rega pelo agendador, botão Lovelace, automações, botão da válvula, ou registe «reguei com a mangueira» — o NeverDry mantém o défice exato em todos os quatro casos.

🛡️

Segurança multicamada

Três camadas de segurança independentes evitam que a válvula fique aberta: watchdog de software (tarefa asyncio), intertravamento de hardware (entidade Zigbee ou MQTT) e limite de tempo de entrega. Se uma camada falhar, a seguinte atua.

Como Funciona

O NeverDry rastreia um balanço hídrico simples para cada zona de rega:

Dzona(t) = Dzona(t-1) + ETh × Kc × Δt − chuva

Quando o défice é alto, o solo está seco e precisa de água. Quando chove, o défice desce. Após a rega, volta a zero. 1 mm de défice = 1 litro por metro quadrado necessário.

Instalação

Via HACS (recomendado)

  1. Abrir HACS → Integrações → Repositórios personalizados
  2. Adicionar https://github.com/drake69/NeverDry como Integração
  3. Procurar NeverDry e instalar
  4. Reiniciar Home Assistant
  5. Definições → Dispositivos e Serviços → Adicionar Integração → NeverDry

Manual

  1. Copiar custom_components/never_dry/ para o diretório config do HA
  2. Reiniciar Home Assistant
  3. Adicionar a integração pela interface

Aviso Legal

O NeverDry é um projeto amador para uso residencial. Não é certificado para aplicações agrícolas, comerciais ou críticas. O modelo ET é uma simplificação da norma FAO-56 e não substitui aconselhamento agronómico profissional. Monitorize sempre o seu sistema de rega.
