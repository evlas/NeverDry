NeverDry

Intelligente Bewässerung für Home Assistant. Ein wissenschaftliches Wasserbilanzmodell, das weiß, wann und wie lange Ihr Garten gegossen werden muss.

GitHub-Repository Dokumentation

Funktionen

🌡️

Wissenschaftliches ET-Modell

Vereinfachte FAO-56-Wasserbilanz. Schätzt die Evapotranspiration aus der Temperatur mit zwei kalibrierbaren Parametern.

🌱

10 Pflanzenfamilien

Pflanzkoeffizient (Kc) pro Zone mit saisonaler Variation. Rasen, Gemüse, Sukkulenten, Obstbäume und mehr. Automatische Hemisphären-Anpassung.

🚿

Direkte Ventilsteuerung

Öffnet und schließt Ventile automatisch. Sequentielle Multi-Zonen-Bewässerung mit Notabschaltung.

📊

Defizit pro Zone

Jede Zone verfolgt ihr eigenes Bodenwasserdefizit. Verschiedene Pflanzen trocknen unterschiedlich schnell aus — so wie es sein soll.

🌧️

Regenberücksichtigung

Überspringt die Bewässerung automatisch an Regentagen. Das Defizit sinkt bei jedem Regenereignis.

⚙️

UI-Konfiguration

Vollständige Einrichtung über die Home-Assistant-Oberfläche. Keine YAML-Bearbeitung erforderlich.

🔄

Nahtlose Updates

Automatisierte Releases über GitHub Actions. Konfigurationsmigration bewahrt Ihre Einstellungen über Versionen hinweg. Ein-Klick-Update in HACS.

🖐️

Manuelles Ventil erkannt

Öffnen Sie das Ventil per Knopf, Zigbee-App oder HA — NeverDry verfolgt die Sitzung, aktualisiert das Defizit und schließt automatisch beim Minimum aus benötigter Wassermenge und Sicherheits-Timeout.

💧

Alle Auslöser abgedeckt

Bewässerung per Scheduler, Lovelace-Button, Automation, Ventilknopf oder »Mit dem Schlauch gegossen« — NeverDry hält das Defizit in allen vier Fällen aktuell.

So funktioniert es

NeverDry verfolgt eine einfache Wasserbilanz für jede Bewässerungszone:

DZone(t) = DZone(t-1) + ETh × Kc × Δt − Regen

Wenn das Defizit hoch ist, ist der Boden trocken und braucht Wasser. Bei Regen sinkt das Defizit. Nach der Bewässerung wird es auf Null zurückgesetzt. 1 mm Defizit = 1 Liter pro Quadratmeter benötigt.

Installation

Über HACS (empfohlen)

  1. HACS öffnen → Integrationen → Benutzerdefinierte Repositories
  2. https://github.com/drake69/NeverDry als Integration hinzufügen
  3. Nach NeverDry suchen und installieren
  4. Home Assistant neu starten
  5. Einstellungen → Geräte & Dienste → Integration hinzufügen → NeverDry

Manuell

  1. custom_components/never_dry/ in Ihr HA-Konfigurationsverzeichnis kopieren
  2. Home Assistant neu starten
  3. Integration über die Oberfläche hinzufügen
